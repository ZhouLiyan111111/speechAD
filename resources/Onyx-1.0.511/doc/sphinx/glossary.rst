.. _glossary:

Glossary
========

.. glossary::
   :sorted:

   branch       
     A copy of a bzr-managed project.

   parent       
     The branch from which another branch of a bzr-managed project is derived.

   working tree 
     The tree of visible, version-controlled files in branch.

   repository   
     The bzr-maintained database of the history of all the
     version-controlled files branch.
