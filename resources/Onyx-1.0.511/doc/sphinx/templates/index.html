Welcome

What users say:

“Cheers for a great tool that actually makes researchers want to write reusable code!”

Onyx is a tool that makes it easy to create intelligent and beautiful experiments. It was originally created to translate the whiteboard sketches of speech scientists into running experiments. It has excellent support for the streaming processing of conceptually unbounded amounts of data for machine-learning and statistical inference applications.

Onyx uses Python as its programming language, and many of its strengths come from the power and simplicity of Python itself, and from Python's high-performance numerical extensions.

Onyx was originally written by Hugh Secker-Walker and Ken Basye at the Johns Hopkins University. Onyx is licensed under the Apache 2 license.
